---
name: grill-me
description: Interview the user relentlessly about a plan or design until reaching shared understanding, resolving each branch of the decision tree. Use when user wants to stress-test a plan, get grilled on their design, or mentions "grill me".
---

Interview me relentlessly about every aspect of this plan until we reach a shared understanding. Walk down each branch of the design tree, resolving dependencies between decisions one-by-one.

For each question:
- Give numbered options (1, 2, 3...) the user can pick from instead of open-ended answers
- Include a short explanation of the relevant background/knowledge so the user understands why each option matters and can make an informed pick
- Mark your recommended option

Ask the questions one at a time.

If a question can be answered by exploring the codebase, explore the codebase instead.
