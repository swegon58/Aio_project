# Product

## Register

product

## Users

Developers and business owners who run repetitive browser, data, and ops tasks. They live in terminals and dashboards. They move fast, hate noise, and want results — not explanations. Primary session: desktop, focused work session, often late-day or late-night.

## Product Purpose

Aio is an AI agent platform that executes real-world tasks — research, automation, document ops — through a conversational interface backed by the Hermes orchestration engine. Success: user types a task, agent runs it, user gets output without friction. The tool disappears into the work.

## Brand Personality

Confident. Direct. Capable.

Voice is terse and concrete: it shows what it did, not what it's about to do. Like a senior engineer who just ships. No exclamation marks in the UI copy. No "Let's get started!" anywhere.

## Anti-references

- manus.im — warm cream bg, generic blue, overly soft. Looks like a template.
- ChatGPT / generic chatbot UIs — centered chat bubble layouts, max-w-[680px] prose, no workspace feeling.
- Notion-style minimal — too editorial, wrong register for an action-oriented tool.
- Purple-gradient SaaS dashboards — noisy, decorative, unfocused.

## Design Principles

1. **The tool disappears.** Every chrome element earns its place by making the task clearer or faster. Nothing decorates.
2. **Show state, not personality.** Motion and color communicate what the agent is doing, not how excited it is. Status first; charm is incidental.
3. **Bold choices, not loud choices.** Strong visual hierarchy without shouting. Dark surface, amber signal, near-white text — committed, not aggressive.
4. **Density is a feature.** Information-dense layouts signal competence. Padding is not respect; whitespace serves rhythm, not safety.
5. **One flow, zero distractions.** The input is the most important element on the screen. Everything else recedes until the agent is active.

## Accessibility & Inclusion

WCAG AA minimum. All interactive states must be keyboard-reachable. Respect `prefers-reduced-motion` for all animations. Amber-on-dark passes contrast; verify on every new color use.
